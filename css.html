Welcome

Welcome to this beginner-friendly CSS course! By the end of these lessons, you’ll be able to style and structure web pages using CSS.

Lesson 1: The Box Model

The CSS Box Model is the foundation of layout design.

element {
    margin: 10px;
    border: 2px solid black;
    padding: 10px;
    width: 200px;
}

Lesson 2: Colors & Backgrounds

CSS allows you to add colors and backgrounds to elements.

body {
    background-color: lightblue;
}
.container {
    background-image: url("background.jpg");
    background-size: cover;
}

Lesson 3: Borders & Shadows

Add borders and shadows to make elements stand out.

.box {
    border: 3px dashed red;
    box-shadow: 5px 5px 10px gray;
}

Lesson 4: CSS Grid Layout

CSS Grid provides a powerful layout system.

.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 10px;
}

Lesson 5: Flexbox

Flexbox makes alignment and spacing easy.

.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Lesson 6: Animations & Transitions

CSS can create animations and smooth transitions.

.animated-box {
    transition: background-color 0.5s ease-in-out;
}
.animated-box:hover {
    background-color: orange;
}

Lesson 7: Responsive Design

Use media queries to make your site responsive.

@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
}

Lesson 8: CSS Positioning

Learn how to position elements on a page.

.relative-box {
    position: relative;
    top: 20px;
    left: 30px;
}

.absolute-box {
    position: absolute;
    top: 50px;
    right: 20px;
}

Lesson 9: Pseudo-classes & Pseudo-elements

Use pseudo-classes to style elements dynamically.

a:hover {
    color: red;
}

p::first-letter {
    font-size: 2em;
    font-weight: bold;
}

Lesson 10: Z-Index

Control the stacking order of elements.

.box1 {
    position: absolute;
    z-index: 1;
}

.box2 {
    position: absolute;
    z-index: 2;
}

Lesson 11: Custom Fonts

Use custom fonts to enhance design.

@font-face {
    font-family: 'MyFont';
    src: url('myfont.woff2') format('woff2');
}

body {
    font-family: 'MyFont', sans-serif;
}
